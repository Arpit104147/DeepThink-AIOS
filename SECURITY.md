# Security Policy

## Supported Versions

We currently support and maintain security updates for the following active version(s) of DeepThink AIOS:

| Version | Supported |
| ------- | --------- |
| main (v1.x) | ✅ Yes |
| < v1.0  | ❌ No |

## Reporting a Vulnerability

As this is an open-source research and developer prototype built by students, we want to maintain a secure environment for everyone running DeepThink AIOS locally. 

If you discover a security vulnerability (such as a sandbox escape, SAST bypass, or command execution vector outside the sandbox boundary), please do not disclose it publicly. 

Instead, report it via one of the following methods:
1. **GitHub Private Vulnerability Reporting:** Go to the "Security" tab of this repository and click "Report a vulnerability" to open a private advisory.
2. **Contact the Maintainers:** Email the project leads directly (please see our profile contact info).

We will investigate the issue and attempt to patch it as soon as possible. Thank you for helping keep this project secure!
